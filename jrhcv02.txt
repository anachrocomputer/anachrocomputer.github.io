1990	INMOS S514 B016 device driver.  Wrote Sun UNIX device driver for INMOS VME transputer board. Flash ROM-based transputer code written in occam, Sun kernel programming in C.

1989	TMI Key Results.  Personal Information Manager program for IBM PC running MS-DOS.  Software written in Borland Turbo Pascal.

1988	Sharp 'Projector'.  Built graphics library in 80x86 assembler for monochrome presentation graphics package.  Code ran on IBM PC with CGA and displayed on LCD panel for overhead projection.

1988	Polaroid 'Shoot'.  Wrote film recorder driver in Microsoft C for IBM PC with EGA card. Assisted with design of composite video converter board.

1987	Enhanced Personal Presentation System.  Presentation graphics package for IBM PC with EGA card.  Most of code in Microsoft BASIC, but also worked on 8086 assembler routines for video special effects and for the image capture TSR.
