Achievements

2002 HP Mobile Bristol project. Built hardware and firmware for wearable computer based on PICmicro chips and I2C bus. Also wrote software in C++ for Windows CE host computer.

2001 Eurologic iSCSI project. Working on software for a RAID disk array suitable for Storage Area Networks (SANs). Code written in ANSI C under the Linux operating system.

1998 Submetrix 2000 Series sonar. Re-implementation of the interferometric sonar using PC-compatible hardware and Windows NT operating system. Embedded real-time programming in C++ with MFC; hardware interface built around QuickLogic FPGA chips.

1996 Submetrix ISIS 100 sonar. Improved shallow-water version of ISIS 300; major hardware and software changes to make the system more compact, reliable and cheaper to manufacture. Worked on embedded real-time software running on transputers and Sun UNIX workstation. Transputer software written in ANSI C; Sun software written in K&R C.

1994 INMOS 'Inquest' debugger. Wrote interrupt-driven device driver running under Microsoft Windows for INMOS transputer motherboard. Code written in 80386 assembler as well as Watcom C.

1992 INMOS S720 X server software. Ported standard MIT X Window System server to the transputer. Also assisted with hardware design and testing for INMOS B020 graphics frame buffer card.

1990 INMOS S514 B016 device driver. Wrote Sun UNIX device driver for INMOS VME transputer board. Flash ROM-based transputer code written in occam, Sun kernel programming in C.

1989 TMI Key Results. Personal Information Manager program for IBM PC running MS-DOS. Software written in Borland Turbo Pascal.

1988 Sharp 'Projector'. Built graphics library in 80x86 assembler for monochrome presentation graphics package. Code ran on IBM PC with CGA and displayed on LCD panel for overhead projection.

1988 Polaroid 'Shoot'. Wrote film recorder driver in Microsoft C for IBM PC with EGA card. Assisted with design of composite video converter board.

1987 Enhanced Personal Presentation System. Presentation graphics package for IBM PC with EGA card. Most of code in Microsoft BASIC, but also worked on 8086 assembler routines for video special effects and for the image capture TSR.
