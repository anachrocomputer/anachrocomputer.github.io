Nintendo N64 and Game Cube Controller Interface

 HOME > Creative Technology > AVR N64 GC

Having successfully connected a Super Nintendo game controller to an AVR microcontroller, I wanted to have a go with a more sophisticated game controller, one with an analog joystick. I chose the Nintendo 64 first of all, and moved on to the (very similar) Game Cube controller. Both these controllers are interfaced to the game console by a three-wire connection, although the Game Cube connector supplies additional power for the vibration motor (the N64 rumble pack contains a battery).
