PlayStation 2 Controller Interface

 HOME > Creative Technology > AVR PSX

I wanted to connect a game control pad to a microcontroller, but I wanted something a little more modern than the Super Nintendo game pad, and something with analog joysticks. (I have another page showing a history of game pad development.) The controller for the Sony PlayStation is interfaced to the game console by an eight-wire connection on a nine-pin plug. The protocol between game console and controller is a bit-serial, bidirectional protocol, and is documented on many web sites with varying degrees of accuracy and completeness. The same protocol is used by the original PlayStation controllers, the NeoCon driving controller, the PlayStation mouse and the light gun.
